This simple project uses Bayesian time series to predict unemployment in the US. We visualize the data using Plotly. Then we repeat the process using a different dataset --- the price of natural gas in the US. I made this project to show how easily we can make powerful time series forecasts without having to get into complicated math.

We begin by importing the python modules we will use. For this project, I use `pandas_datareader` to fetch data from the [Federal Reserve Economic Data | FRED](https://fred.stlouisfed.org/).
